About me

Nama saya Idris Fadilah dan sering dipanggil Idris. Saya lahir di Serang, 17 juni 2002. Saya adalah orang yang sangat antusias dengan segala hal berbau pemrograman dan selalu merasa tertantang untuk mengerjakan projek-projek baru pembuatan aplikasi berbasis web atau mobile.

Saya pun merasa bahagia selama di Universitas Banten Jaya. Meskipun kadang ada rasa jenuh, saya tetap bertahan karena bagi saya, itu adalah ujian bagi seorang Mahasiswa. Untuk membuang jenuh, saya memilih untuk membaca banyak buku dan menonton YouTube :v.

My experience

Perkenalkan saya Idris Fadilah dari mahasiswa Universitas Banten Jaya jurusan Teknik Informatika. Saya mulai coding di masa kuliah saya dan pada saat saya berusia 17 tahun. Dan saya seorang Coder yang biasa-biasa saja. Tetapi saya yakin bahwa saya dapat mempelajari bahasa baru, jika diperlukan, dan mulai menulis beberapa kode "biasa-biasa saja" dalam bahasa itu dalam sebulan. Karena semua hal yang perlu saya ketahui tentang bahasa apa pun, saya sudah tahu. Maksud saya dasar-dasar bahasa pemrograman. Saya cukup beruntung memiliki seseorang yang memastikan bahwa saya mempelajari dasar-dasarnya dengan benar.
